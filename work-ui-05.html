UI 设计

新媒体营销理实一体化实训系统

基于真实岗位需求的沉浸式 B 端实战训练平台

场景化实训平台 短视频运营实操 仿真业务流交互 理实一体化体验

项目概述

设计的核心挑战：如何将庞杂枯燥的理论知识与高度复杂的短视频运营实操工具，封装进一个统一且富有带入感的视觉框架中，让学生在操作时能感受到真实工作环境的紧张感与成就感，实现"理与实"的平滑过渡。

设计亮点

剧情化的沉浸式场景构建

突破传统的表单与列表展示，通过任务流设计，将学习过程包装为真实的"职业挑战任务"，提升了职校学生的角色代入感。

高拟真"生产者视角"实训工具链

针对新媒体营销专业的核心技能需求，设计了具备深度交互体验的"互动网页"与"实训工具"UI。建立了一套带有明确可视化反馈（如状态标签、微动效）的工作流界面，让复杂的账号管理、内容排版、定时发布等真实业务操作清晰易用。

我的角色

独立视觉设计师。负责全站所有页面的 UI 界面设计，将复杂的"仿真剧情"与"实操工具"逻辑转化为高效、直观且极具代入感的交互与视觉架构。
