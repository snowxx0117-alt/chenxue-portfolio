联创新世纪职教官网

职业教育品牌官网全站 UI 设计

B端平台官网 复杂信息架构 系统级 UI 组件库

项目概述

这是一个涵盖 1+X 证书、技能大赛、师资培训等众多垂直领域的信息密集型 B 端平台。设计的核心挑战在于梳理庞杂的业务线，在海量数据中建立清晰的视觉秩序与操作逻辑。

设计亮点

模块化信息架构

运用严谨的网格与卡片化设计，对繁杂的业务内容进行收纳与重组，有效降低了用户的阅读与认知负担。

标准化组件沉淀

为保障多业务线的视觉统一，搭建了一套基础 UI 组件规范（涵盖导航、列表、数据图表等），提升了设计的复用率。

专业的视觉基调

平衡了 B 端平台的信息密度与视觉呼吸感，以浅色调为主，通过清晰的层级划分，提升重度业务信息的阅读体验。

我的角色

独立视觉设计师。负责该平台全链路的视觉交互与 UI 设计工作。深入梳理了复杂的业务层级，建立并应用了全站的 UI 组件规范，独立完成了所有业务页面的视觉交付。
