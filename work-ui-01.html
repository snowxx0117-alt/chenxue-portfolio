UI 设计

联创新世纪官网

品牌官网全站视觉系统设计与落地

企业主站设计 视觉规范定义 多业务线整合 全屏视觉排版 多端响应式适配

项目概述

作为联创新世纪的品牌数字大本营，该项目涵盖了企业主站与核心的"品牌营销"业务子站。设计的重点在于构建一个包容性强的视觉框架，既能展示宏观的企业愿景与实力背书，又能自然衔接并精准传递品牌营销业务的创意服务价值。

设计亮点

主次分明的矩阵架构

以沉浸式的主站页面作为视觉基底，自然平滑地引导至"品牌营销"垂直业务线，构建了清晰的用户浏览与商业转化路径。

稳健与创意的视觉平衡

在企业主站保持深色沉稳基调的同时，品牌营销业务页则运用了更具视觉冲击力的排版与色彩张力，精准契合了创意营销的业务属性。

连贯的沉浸式体验

通过全局优化的空间留白与纵深感排版，打破常规企业站点的刻板印象，确保用户在主站与各业务子站间穿梭时，能获得连贯、舒适的阅读节奏。

我的角色

独立视觉设计师。全面负责主站及品牌营销子站的 UI 界面设计与最终落地。从品牌视觉语言的提取转化，到各业务线之间"视觉一致性与差异化"的精细把控，保障了企业形象与核心业务在数字端的高水准呈现。
